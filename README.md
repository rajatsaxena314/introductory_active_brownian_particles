# Active Brownian Particles: Simulation and Fokker–Planck Analysis

This repository contains numerical simulations and analysis tools for studying **Active Brownian Particles (ABPs)**. 
The project is designed for exploratory research and numerical experimentation with stochastic particle systems.

The topics appear in this repo as they do in [Active Brownian Particles. From Individual to Collective Stochastic Dynamics](https://arxiv.org/abs/1202.2442)

## Overview

Active Brownian particles are self-propelled particles whose motion is influenced by both deterministic active motion and stochastic fluctuations. Their trajectories often show a combination of directed motion, diffusion, persistence, and noise-driven spreading.
